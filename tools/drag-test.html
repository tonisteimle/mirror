Drag & Drop Browser Tests

How to use this test page

This page must be opened inside the Mirror Studio to access the __dragTest API.

To run tests:

  1. Start the Studio: npm run studio
  2. Open the Studio at http://localhost:5173/studio/
  3. Open browser console (F12)
  4. Run tests manually: __dragTest.runAllTests()

Or use the CLI tool: npx tsx tools/drag-test-runner.ts
