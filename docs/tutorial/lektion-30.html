Slots

Überblick

Slots ermöglichen flexible Komponenten, bei denen Teile des Inhalts von außen eingefügt werden können. Das ist die Grundlage für wiederverwendbare Layout-Komponenten.

Übung

Erstelle eine Alert-Komponente mit Slots für Icon, Nachricht und Aktion:

Lösung 
Alert: = Frame hor, gap 12, pad 12, bg #1a1a1a, rad 8
  Frame w 24, center
    Slot "icon"
  Frame grow
    Slot "message"
  Slot "action"

Frame gap 8
  Alert
    slot "icon"
      Icon "info", ic #2563eb, is 20
    slot "message"
      Text "Neue Updates verfügbar.", col white, fs 14
    slot "action"
      Button "Update", bg #2563eb, col white, pad 6 12, rad 4, fs 12

  Alert
    slot "icon"
      Icon "check-circle", ic #10b981, is 20
    slot "message"
      Text "Erfolgreich gespeichert!", col white, fs 14

  Alert
    slot "icon"
      Icon "alert-triangle", ic #f59e0b, is 20
    slot "message"
      Text "Speicher fast voll.", col white, fs 14
    slot "action"
      Text "Bereinigen", col #f59e0b, fs 12, cursor pointer

Zusammenfassung

  • Slot – Platzhalter für Kinder-Elemente
  • Slot "name" – Benannter Platzhalter
  • slot "name" – Zuweisung zu benanntem Slot
  • Default-Inhalt möglich als Fallback
  • Ideal für: Layouts, Dialoge, Listen, wiederverwendbare UI
