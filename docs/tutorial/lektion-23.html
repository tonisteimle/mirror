Enter/Exit Animationen

Überblick

Enter- und Exit-Animationen spielen beim Erscheinen und Verschwinden von Elementen ab. Sie machen State-Übergänge flüssiger und professioneller.

enter: animationAnimation beim Erscheinen
exit: animationAnimation beim Verschwinden
onenter:Event wenn Element erscheint
onexit:Event wenn Element verschwindet

Übung

Erstelle eine Bestätigungs-Dialog mit Backdrop, wobei der Backdrop fade-in/out und das Modal scale-in/out macht:

Lösung 
Frame w 300, h 180, stacked, bg #0c0c0c
  // Trigger
  Button pad 12 20, bg #ef4444, col white, rad 8, cursor pointer, pc
    onclick:
      open Dialog
    "Löschen"

  // Backdrop
  Frame w full, h full, bg rgba(0,0,0,0.5)
    visible when Dialog open
    enter: fade-in
    exit: fade-out

  // Dialog
  Frame w 200, bg #1a1a1a, rad 12, pad 20, pc, z 10
    visible when Dialog open
    enter: scale-in
    exit: scale-out
    Text "Wirklich löschen?", col white
    Frame hor, gap 8
      Button pad 8 12, bg #333, col white, rad 6, cursor pointer
        onclick:
          close Dialog
        "Abbrechen"
      Button pad 8 12, bg #ef4444, col white, rad 6, cursor pointer
        onclick:
          close Dialog
        "Löschen"

Zusammenfassung

  • enter: animation – Animation beim Erscheinen
  • exit: animation – Animation beim Verschwinden
  • Funktioniert mit: fade-in/out, slide-in/out, scale-in/out, bounce
  • Kombinierbar mit visible when
  • Ideal für: Modals, Dropdowns, Toasts, Sidebars
