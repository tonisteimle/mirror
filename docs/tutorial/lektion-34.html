Überblick

page "name"Navigiert zu einer Seite
navigate "url"Navigiert zu URL

Übung

Erstelle eine einfache Onboarding-Sequenz mit 3 Screens:

Lösung 
$onboardingStep: 1

Frame w 260, h 200, bg #1a1a1a, rad 12, pos
  // Screen 1
  Frame center, gap 12, pad 20
    visibleWhen $onboardingStep == 1
    Icon "star", ic #f59e0b, is 40
    Text "Willkommen!", col white, weight 500, fs 18
    Text "Schön, dass du da bist.", col #888, fs 13

  // Screen 2
  Frame center, gap 12, pad 20
    visibleWhen $onboardingStep == 2
    Icon "zap", ic #2563eb, is 40
    Text "Schnell & Einfach", col white, weight 500, fs 18
    Text "Erstelle UIs in Sekunden.", col #888, fs 13

  // Screen 3
  Frame center, gap 12, pad 20
    visibleWhen $onboardingStep == 3
    Icon "check-circle", ic #10b981, is 40
    Text "Bereit!", col white, weight 500, fs 18
    Text "Los geht's!", col #888, fs 13

  // Progress Dots
  Frame hor, gap 6, center, absolute, pb 50, pcx
    Frame w 8, h 8, rad 4
      bg #2563eb
      visibleWhen $onboardingStep == 1
    Frame w 8, h 8, rad 4, bg #333
      visibleWhen $onboardingStep != 1

    Frame w 8, h 8, rad 4
      bg #2563eb
      visibleWhen $onboardingStep == 2
    Frame w 8, h 8, rad 4, bg #333
      visibleWhen $onboardingStep != 2

    Frame w 8, h 8, rad 4
      bg #2563eb
      visibleWhen $onboardingStep == 3
    Frame w 8, h 8, rad 4, bg #333
      visibleWhen $onboardingStep != 3

  // Navigation
  Frame hor, spread, pad 16, absolute, pb 12, pl 12, pr 12
    Button "Zurück", pad 8 16, rad 6, col #888, cursor pointer
      onclick: page "prev"
      visibleWhen $onboardingStep > 1
    Frame grow
    Button "Weiter", pad 8 16, bg #2563eb, col white, rad 6, cursor pointer
      onclick: page "next"
      visibleWhen $onboardingStep < 3
    Button "Starten", pad 8 16, bg #10b981, col white, rad 6, cursor pointer
      visibleWhen $onboardingStep == 3

Zusammenfassung

  • page "name" – Wechselt zu einer Ansicht
  • navigate "url" – Öffnet externe URL
  • $currentPage – State für aktuelle Seite
  • visibleWhen $page == "x" – Bedingte Anzeige
  • Ideal für: Tabs, Wizards, Sidebars, Mobile Nav
