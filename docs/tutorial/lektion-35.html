API-Integration

Mirror-Code kann mit JavaScript für API-Aufrufe kombiniert werden:

// JavaScript
async function fetchUsers() {
  const response = await fetch('/api/users');
  const users = await response.json();
  // Update Mirror state
  mirrorApp.setState('users', users);
}

// Mirror
Button "Laden", pad 12 24, bg #2563eb, col white, rad 8
  onclick: call fetchUsers()

Frame gap 8
  each user in $users
    Text user.name, col white

Der kompilierte Code gibt dir Zugriff auf State-Management-Funktionen.

Compiler API

So verwendest du den Mirror-Compiler in JavaScript:

// Mirror-Code kompilieren
const mirrorCode = `
Button "Klick mich", bg #2563eb, col white, pad 12 24, rad 8
`;

// Zu JavaScript kompilieren
const jsCode = MirrorLang.compile(mirrorCode);

// Code ausführen
const createUI = new Function(jsCode + '\nreturn createUI();');
const ui = createUI();

// DOM einfügen
document.getElementById('app').appendChild(ui.root);

React-Integration

Mirror kann auch React-Komponenten generieren:

import { compileMirror } from 'mirror-lang';

// Mirror zu React kompilieren
const reactCode = compileMirror(mirrorCode, { target: 'react' });

// Als React-Komponente verwenden
function App() {
  const MirrorComponent = useMemo(() => {
    return compileMirror(`
      Frame gap 12
        Text "React + Mirror", col white, weight 500
        Button "Action", bg #2563eb, col white, pad 10 20, rad 8
    `);
  }, []);

  return <MirrorComponent />;
}

Praxisbeispiel: Timer

Ein Timer mit JavaScript-Integration:

In einer echten App würde setInterval den Timer aktualisieren:

let interval;

function startTimer() {
  interval = setInterval(() => {
    if (app.getState('isRunning')) {
      app.setState('seconds', app.getState('seconds') + 1);
    }
  }, 1000);
}

function stopTimer() {
  clearInterval(interval);
}

Build-Integration

Mirror in Build-Tools integrieren:

// vite.config.js
import { mirrorPlugin } from 'mirror-lang/vite';

export default {
  plugins: [
    mirrorPlugin({
      // .mirror Dateien automatisch kompilieren
      include: '**/*.mirror',
      target: 'dom' // oder 'react'
    })
  ]
};

// Dann in deinem Code:
import MyComponent from './MyComponent.mirror';

document.body.appendChild(MyComponent().root);
