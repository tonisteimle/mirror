Custom States

Was sind States?

In Lektion 4 hast du hover:, focus: und disabled: kennengelernt – das sind System-States. Jetzt lernst du, wie du eigene States definierst.

Custom States ermöglichen interaktive UIs: Tabs die sich umschalten, Checkboxen die ihren Zustand ändern, Menüs die auf- und zuklappen.

State definieren

Verfügbare Custom States

Mirror bietet vordefinierte State-Namen für häufige Anwendungsfälle:

state selectedFür Auswahl (Tabs, Liste)
state highlightedFür Hervorhebung
state expandedFür auf-/zuklappbare Bereiche
state collapsedGegenteil von expanded
state openFür Dialoge, Dropdowns
state closedGegenteil von open
state onFür Toggles, Switches
state offGegenteil von on

Zusammenfassung

  • state selected – Auswahl-State
  • state expanded – Aufgeklappt-State
  • state on – Toggle-State (an)
  • state off – Toggle-State (aus)
  • state open – Offen-State
  • state closed – Geschlossen-State
  • State-Block: selected:, on:, etc.
  • Im State-Block: Properties überschreiben
