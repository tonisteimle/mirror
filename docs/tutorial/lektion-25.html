Event Modifiers

Überblick

Event Modifiers verändern, wie Events verarbeitet werden – zum Beispiel mit Verzögerung oder Debouncing.

debounce NWartet N ms nach letztem Event
delay NVerzögert Ausführung um N ms
onclick-outside:Klick außerhalb des Elements

Debounce

debounce wartet, bis der Benutzer aufhört zu tippen, bevor die Action ausgeführt wird:

Debounce ist ideal für Suchen – die Anfrage wird erst gesendet, wenn der Benutzer kurz pausiert.

Zusammenfassung

  • debounce N – Wartet N ms nach letztem Event
  • delay N – Verzögert Ausführung um N ms
  • onclick-outside: – Klick außerhalb des Elements
  • Debounce ideal für: Suche, Auto-Save, Validierung
  • Delay ideal für: Tooltips, Animationssequenzen
  • Click-Outside ideal für: Dropdowns, Modals, Popover
