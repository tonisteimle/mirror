Effekte

Schatten

shadow fügt einen Schlagschatten hinzu. Es gibt drei Presets:

shadow sm Kleiner Schatten
shadow md Mittlerer Schatten
shadow lg Großer Schatten

Transparenz

opacity macht Elemente transparent. 1 = voll sichtbar, 0 = unsichtbar:

Transparenz ist nützlich für Hover-Effekte und deaktivierte Elemente:

Blur

blur macht ein Element unscharf:

Backdrop Blur

backdrop-blur (oder blur-bg) macht den Hintergrund hinter dem Element unscharf. Perfekt für Glaseffekte:

    Text "Projekt Alpha", col white, weight 500
    Text "Neu", bg #2563eb, col white, pad 2 8, rad 4, fs 12, opacity 0.8
  Text "Aktiv seit 3 Tagen", col #888, fs 14

Zusammenfassung

  • shadow sm/md/lg – Schlagschatten
  • opacity N – Transparenz (0-1)
  • blur N – Element unscharf
  • backdrop-blur N – Hintergrund unscharf
