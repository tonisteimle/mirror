Each-Loops

Überblick

Mit each iterierst du über Daten-Collections und erzeugst für jedes Element UI-Komponenten. Das ist die Grundlage für dynamische Listen, Grids und Tabellen.

Zusammenfassung

  • each item in Collection – Iteration über Daten
  • each item, index in Collection – Mit Index
  • item.property – Zugriff auf Properties
  • Verschachtelung möglich für komplexe Daten
  • Funktioniert mit allen Layouts (ver, hor, grid)
