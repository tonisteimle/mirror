Positioned Layout

Stacked Layout

Standardmäßig werden Kinder untereinander (ver) oder nebeneinander (hor) angeordnet. Mit stacked liegen alle Kinder übereinander:

Alle drei Frames liegen übereinander. Das zuletzt definierte Element liegt oben.

Position mit x und y

x und y verschieben ein Element relativ zu seiner normalen Position:

Z-Index

z kontrolliert die Stapel-Reihenfolge. Höhere Werte liegen weiter oben:

Blau (z 3) liegt ganz oben, dann Orange (z 2), dann Grün (z 1).

Positioned Container

pos macht einen Container zum Bezugspunkt für absolut positionierte Kinder:

Die roten und blauen Frames sind absolute positioniert relativ zum Container mit pos.

Absolute vs. Fixed vs. Relative

Drei Positionierungs-Modi:

relativeVerschiebt relativ zur normalen Position
absolutePositioniert relativ zum nächsten pos-Container
fixedPositioniert relativ zum Viewport (bleibt beim Scrollen)

Zusammenfassung

  • stacked – Kinder übereinander stapeln
  • pos – Container für absolute Positionierung
  • x N, y N – Position verschieben
  • z N – Stapel-Reihenfolge (höher = weiter oben)
  • absolute – Relativ zum pos-Container
  • fixed – Relativ zum Viewport
  • relative – Relativ zur normalen Position
  • rotate N – Element drehen (Grad)
