Data Binding

Überblick

Data Binding verbindet UI-Elemente mit Daten. Änderungen an den Daten aktualisieren automatisch die UI – und umgekehrt.

data CollectionBindet Komponente an Daten
selection $variableSpeichert ausgewähltes Element
value $variableBindet Input-Wert an Variable

Übung

Erstelle einen Farbwähler mit Vorschau der ausgewählten Farbe:

Lösung 
Colors:
  - name: "Rot", hex: "#ef4444"
  - name: "Grün", hex: "#10b981"
  - name: "Blau", hex: "#2563eb"
  - name: "Lila", hex: "#8b5cf6"

Frame gap 12, data Colors, selection $color
  // Farbliste
  Frame hor, gap 8
    each color in Colors
      Frame state selected exclusive, w 40, h 40, rad 8, cursor pointer
        bg color.hex
        selected:
          bor 3, boc white

  // Vorschau
  Frame hor, gap 12, pad 12, bg #1a1a1a, rad 8
    Frame w 40, h 40, rad 8
      bg $color.hex
    Frame gap 2
      Text $color.name, col white, weight 500
      Text $color.hex, col #888, fs 12

Zusammenfassung

  • data Collection – Bindet an Daten-Collection
  • selection $variable – Speichert Auswahl
  • value $variable – Zwei-Wege-Binding für Inputs
  • $variable.property – Zugriff auf Auswahl-Properties
  • Ideal für: Listen, Formulare, Master-Detail
