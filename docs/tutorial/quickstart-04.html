Hover, States und Events

UIs müssen auf Benutzer reagieren. In Mirror gibt es dafür drei Mechanismen: hover: für Maus-Effekte, state für eigene Zustände (an/aus, selektiert, etc.), und Events wie onclick: für explizite Aktionen.

Custom States

Mit state definierst du eigene Zustände. Elemente mit einem State sind automatisch klickbar – ein Klick wechselt den Zustand. Der selected:-Block definiert das Styling im aktiven Zustand:

exclusive bedeutet: Nur eines kann gleichzeitig aktiv sein – wie Radio-Buttons.
