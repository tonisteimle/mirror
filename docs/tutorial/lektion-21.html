Hover-Transitions

Die einfachste Art, Animationen zu nutzen, sind Hover-Effekte. Die Änderungen im hover: Block werden automatisch sanft animiert:

Fahre mit der Maus über die Buttons – die Farb- und Größenänderungen werden sanft animiert.

Hover-Properties

Es gibt spezielle Hover-Properties für schnelle Effekte:

hover-bgHintergrund bei Hover
hover-colTextfarbe bei Hover
hover-opacityTransparenz bei Hover
hover-scaleSkalierung bei Hover
hover-borBorder bei Hover
hover-bocBorder-Farbe bei Hover
hover-radRadius bei Hover

Übung

Erstelle einen Favoriten-Button, der bei Hover gelb wird und bei Klick den "on" State aktiviert:

Lösung 
Frame state on, hor, gap 8, pad 10 20, bg #333, rad 8, cursor pointer
  Icon "star", ic #888, is 18
  Text "Favorit", col #888
  hover:
    bg #444
    Icon "star", ic #f59e0b
    Text "Favorit", col #f59e0b
  on:
    bg #f59e0b
    Icon "star", ic white, fill
    Text "Favorit", col white

Zusammenfassung

  • hover: – Block für Hover-Änderungen (automatisch animiert)
  • hover-bg – Kurzform für Hintergrund-Transition
  • hover-col – Kurzform für Textfarben-Transition
  • hover-scale – Kurzform für Skalierungs-Transition
  • hover-opacity – Kurzform für Transparenz-Transition
  • Mehrere Properties können gleichzeitig animiert werden
  • Transitions funktionieren auch mit States
