Zusammenfassung

  • grid N – Grid mit N Spalten
  • grid auto – Automatische Spaltenanzahl
  • gap-x gap-y – Unterschiedliche Abstände
  • dense – Lücken füllen
  • Grid = gleichmäßig, Flex = nach Inhalt
