What is Mirror?

Mirror is a design-first language for creating user interfaces. Write simple, readable code and see your designs come to life instantly. Perfect for designers who want to prototype without learning complex frameworks.

Chapter 1

Your First UI

Let's create your first Mirror interface. You'll learn how to create elements, add text, and apply basic styling.

Elements

In Mirror, everything starts with an element. The most basic element is a Frame - think of it as a container or a box.

lightbulb
Try it! Change the text inside the quotes. The preview updates instantly.

Nesting Elements

Elements can contain other elements. Use indentation (2 spaces) to show that something is inside another element.

Notice how both Text elements are indented inside the Frame. That's how Mirror knows they belong inside.

Common Properties

Properties change how elements look. Here are the most common ones:

bg Background color bg #3b82f6
col Text color col white
pad Padding (inner space) pad 16
rad Border radius (rounded corners) rad 8
fs Font size fs 24
weight Font weight weight bold

Other Elements

Besides Frame and Text, Mirror has many other elements:

info
Lines starting with // are comments. They help explain your code but don't show up in the preview.
