Mirror

Lerne Mirror

Eine Sprache für UIs – für Designer, nicht nur für Entwickler.

HTML und CSS waren nie für Anwendungen gedacht. Deshalb entstanden Frameworks wie React, Vue und Angular. Doch diese richten sich an Entwickler – mit komplexer Syntax, Build-Tools und Boilerplate.

Mirror ist anders. Eine Beschreibungssprache, die radikal einfach ist. Tokens definieren Werte, Komponenten nutzen Tokens, Layouts enthalten Komponenten. Kein Overhead, keine Magie.
