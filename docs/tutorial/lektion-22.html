State Dependencies

Überblick

State Dependencies erlauben es, dass Elemente auf den State anderer Elemente reagieren. Mit visible when zeigst du Elemente nur, wenn bestimmte Bedingungen erfüllt sind.

visible when Target stateSichtbar wenn Target den State hat
when ... or ...Sichtbar bei einem von mehreren
when ... and ...Sichtbar bei allen Bedingungen

Frame w 240, bg #1a1a1a, rad 8, clip
  // Section 1 Header

    bor 0 0 1 0, boc #333, cursor pointer
    Text "Section 1", col #888

      Text "Section 1", col white

    Text "Content 1 hier...", col #888, fs 13

  // Section 2 Header
  Frame state expanded exclusive, hor, spread, pad 12, bg #252525
    cursor pointer
    Text "Section 2", col #888
    Icon "chevron-right", ic #888, is 14
    expanded:
      Text "Section 2", col white
      Icon "chevron-down", ic #888, is 14
  // Section 2 Content
  Frame pad 16
    visible when Section2 expanded
    Text "Content 2 hier...", col #888, fs 13

Zusammenfassung

  • visible when Target state – Sichtbar wenn Bedingung erfüllt
  • when A or B – Sichtbar wenn A oder B
  • when A and B – Sichtbar wenn A und B
  • Ideal für: Dropdowns, Tabs, Modals, Validierung
  • Ermöglicht komplexe UI-Interaktionen ohne Code
