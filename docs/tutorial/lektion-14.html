Pinning

Pin zu den Kanten

Pin-Properties heften Elemente an bestimmte Kanten eines pos-Containers. Der Wert ist der Abstand zur Kante:

pin-left NAbstand zur linken Kante (Alias: pl)
pin-right NAbstand zur rechten Kante (Alias: pr)
pin-top NAbstand zur oberen Kante (Alias: pt)
pin-bottom NAbstand zur unteren Kante (Alias: pb)

Zentriertes Pinning

Mit pin-center-x, pin-center-y oder pin-center zentrierst du Elemente:

pin-center-xHorizontal zentriert (Alias: pcx)
pin-center-yVertikal zentriert (Alias: pcy)
pin-centerBeides (Alias: pc)

Zusammenfassung

  • pin-left N / pl N – Abstand zur linken Kante
  • pin-right N / pr N – Abstand zur rechten Kante
  • pin-top N / pt N – Abstand zur oberen Kante
  • pin-bottom N / pb N – Abstand zur unteren Kante
  • pin-center-x / pcx – Horizontal zentriert
  • pin-center-y / pcy – Vertikal zentriert
  • pin-center / pc – Komplett zentriert
  • Zwei gegenüberliegende Pins = Element wird gestreckt
