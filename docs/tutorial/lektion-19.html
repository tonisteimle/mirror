Actions

Überblick

Actions sind Befehle, die auf Events folgen. Sie ändern States, zeigen/verstecken Elemente oder navigieren.

show TargetElement sichtbar machen
hide TargetElement verstecken
toggle TargetSichtbarkeit umschalten
open TargetState "open" aktivieren
close TargetState "open" deaktivieren
select TargetState "selected" aktivieren
activate TargetElement aktivieren
deactivate TargetElement deaktivieren

Zusammenfassung

  • show Target – Element anzeigen
  • hide Target – Element verstecken
  • toggle Target – Sichtbarkeit umschalten
  • open Target – "open" State setzen
  • close Target – "open" State entfernen
  • select Target – "selected" State setzen
  • highlight next/prev/first/last – Navigation
  • focus Target – Fokus setzen
  • blur Target – Fokus entfernen
  • submit Target – Formular absenden
  • reset Target – Formular zurücksetzen
  • self – Bezieht sich auf das eigene Element
