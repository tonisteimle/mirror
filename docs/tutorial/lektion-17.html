State-Modifikatoren

Überblick

State-Modifikatoren steuern, wie sich States verhalten: ob sie sich gegenseitig ausschließen, toggeln oder initial aktiv sind.

exclusiveNur einer in der Gruppe kann aktiv sein
toggleKlick schaltet State um (an/aus)
initialState ist beim Start aktiv

Zusammenfassung

  • exclusive – Nur einer in der Gruppe aktiv
  • initial – State beim Start aktiv
  • toggle – Explizites Toggle-Verhalten
  • Tabs: state selected exclusive
  • Checkboxen: state on (unabhängig)
  • Radios: state selected exclusive
