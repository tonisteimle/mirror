Übung

Erstelle eine Warenkorb-Anzeige mit Preis und "Leer"-Zustand:

Lösung 
Cart:
  items:
    - name: "Laptop", price: 999
    - name: "Mouse", price: 49
  total: 1048

Frame w 240, bg #1a1a1a, rad 8, pad 16, gap 12
  Frame hor, spread
    Text "Warenkorb", col white, weight 500
    if Cart.items.length > 0
      Frame pad 4 10, bg #2563eb, rad 12
        Text Cart.items.length, col white, fs 11

  if Cart.items.length > 0
    each item in Cart.items
      Frame hor, spread, pad 8
        Text item.name, col white, fs 13
        Text item.price + " €", col #888, fs 13

    Frame hor, spread, pad 8, bor 1 0 0 0, boc #333
      Text "Gesamt", col white, weight 500
      Text Cart.total + " €", col #10b981, weight 500
  else
    Frame center, pad 24, gap 8
      Icon "shopping-cart", ic #555, is 32
      Text "Warenkorb ist leer", col #666, fs 13

Zusammenfassung

  • if condition – Bedingte Anzeige
  • else – Alternative wenn falsch
  • visibleWhen condition – Inline-Sichtbarkeit
  • Vergleiche: ==, !=, >, <
  • Ideal für: Leere Zustände, Rollen, Status, Validierung
