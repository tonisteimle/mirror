Keyboard Events

Überblick

Keyboard Events ermöglichen die Steuerung per Tastatur. Das ist essentiell für Accessibility und Power-User.

onkeydown key:Bei Tastendruck
onkeyup key:Bei Tastenfreigabe

Übung

Erstelle ein Modal, das mit Escape geschlossen und mit Enter bestätigt werden kann:

Lösung 
Frame w 300, h 160, stacked, bg #0c0c0c
  Button pad 12 20, bg #ef4444, col white, rad 8, cursor pointer, pc
    onclick:
      open Dialog
    "Löschen"

  Frame w 220, bg #1a1a1a, rad 12, pad 20, pc, focusable
    visible when Dialog open
    onkeydown escape:
      close Dialog
    onkeydown enter:
      close Dialog
    Text "Wirklich löschen?", col white, weight 500
    Frame hor, gap 8
      Button pad 8 12, bg #333, col white, rad 6, cursor pointer
        onclick:
          close Dialog
        "Abbrechen"
      Button pad 8 12, bg #ef4444, col white, rad 6, cursor pointer
        onclick:
          close Dialog
        "Löschen"

Zusammenfassung

  • onkeydown key: – Reagiert auf Tastendruck
  • escape – Zum Schließen von Dialogen
  • enter – Zum Bestätigen
  • space – Zum Toggeln
  • arrow-up/down – Für Listen-Navigation
  • home/end – Zum Anfang/Ende springen
  • focusable – Macht Element fokussierbar
