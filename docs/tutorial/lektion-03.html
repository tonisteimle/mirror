Übung

Refaktoriere diesen Code mit Tokens. Die Farben sollen zentral definiert sein:

Lösung 
$bg: #18181b
$accent: #2563eb
$text: #ffffff
$muted: #888888

Frame bg $bg, pad 24, rad 12, gap 12
  Text "Dashboard", col $text, fs 20, weight bold
  Text "Willkommen zurück", col $muted
  Button "Neues Projekt", bg $accent, col $text, pad 12 24, rad 6

Zusammenfassung

  • $name: wert – Token definieren
  • bg $name – Token verwenden
  • Farben, Abstände, Größen – alles tokenisierbar
  • Tokens + Komponenten = Design-System
