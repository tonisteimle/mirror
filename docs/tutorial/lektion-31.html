Vererbung

Überblick

Mit Vererbung erstellst du Varianten von Komponenten. Die Kind-Komponente erbt alle Eigenschaften der Eltern-Komponente und kann diese überschreiben oder erweitern.

Übung

Erstelle ein Badge-System mit Basis und Farbvarianten (Success, Warning, Error):

Lösung 
// Basis-Badge
Badge: = Frame pad 4 10, rad 12, bg #374151
  Text "Badge", col white, fs 11, weight 500

// Farbvarianten
SuccessBadge as Badge: = bg #10b981
WarningBadge as Badge: = bg #f59e0b
ErrorBadge as Badge: = bg #ef4444
InfoBadge as Badge: = bg #2563eb

// Mit Outline-Variante
OutlineBadge as Badge: = bg transparent, bor 1, boc #555
  Text "Badge", col #888

// Test
Frame gap 12
  Frame hor, gap 8
    Badge "Default"
    SuccessBadge "Success"
    WarningBadge "Warning"
    ErrorBadge "Error"
    InfoBadge "Info"
  Frame hor, gap 8
    OutlineBadge "Outline"

Zusammenfassung

  • Child as Parent: = – Erbt von Eltern-Komponente
  • Alle Properties werden vererbt
  • Kind kann Properties überschreiben
  • States werden mitvererbt
  • Ideal für: Button-Systeme, Input-Varianten, Cards
