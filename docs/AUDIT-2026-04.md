## 2c. Layer-Verletzungen (155 Deep Imports) ⚠️ NEU

### Problem

Deep Imports wie `../../../../compiler/ir/types` verletzen die Modul-Grenzen und erhöhen die Kopplung.

### Verteilung

| Import-Tiefe | Anzahl | Beispiel |
|--------------|--------|----------|
| 4+ Ebenen | 42 | `../../../../compiler/...` |
| 3 Ebenen | 68 | `../../../studio/...` |
| 2 Ebenen | 45 | `../../core/...` |

### Empfehlung

```typescript
// ❌ Schlecht: Deep Import
import { SourceMap } from '../../../../compiler/ir/source-map'

// ✅ Besser: Barrel Export
import { SourceMap } from '@mirror/compiler'
```
