Komponenten definieren

Mit Name: (Doppelpunkt) wird eine wiederverwendbare Komponente definiert:

// Definition Button: hor cen cen h 40 pad l r 16 rad 8 bg #3B82F6 col #FFF // Verwendung Button "Klick mich" Button i "plus" "Neu"
